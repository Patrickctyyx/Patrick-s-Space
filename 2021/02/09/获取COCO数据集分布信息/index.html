获取 COCO 数据集分布信息

patrickcty 2月 09, 2021
  • COCO
  • Python
  •  浏览量
  • 分享到微博

使用 COCO API 来获得数据集中的数据分布信息，参考CSDN 文章。

from pycocotools.coco import COCO

coco=COCO(PATH_TO_COCO_JSON)

# 获取所有类别对应的名称与 id，一共有 80 类，id 号则是 1~90
cat_info = [(c['id'], c['name']) for c in coco.cats.values()]

for cat_id, cat_name in cat_info:
    anns = coco.catToImgs[cat_id]
    print("{} | {} | {:.2%} | {} | {:.2%}".format(cat_name, len(set(anns)), len(set(anns)) / len(coco.imgs), len(anns), len(anns) / len(coco.anns)))
